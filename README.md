# Frutos de Goiás - Mapa de Estabelecimentos

Projeto desenvolvido para processamento de dados e visualização geográfica.

## Estrutura do Projeto

* `Mapa_Estabelecimentos_05.py`: Script principal desenvolvido em Python, responsável pela lógica de dados e geração do mapa.
* `index.html`: Arquivo de interface web estática para apresentação visual interativa dos resultados gerados[cite: 3].

## Pré-requisitos

Certifique-se de ter instalado em sua máquina:
* Python 3.x
* Navegador Web atualizado (Chrome, Firefox, Edge)

## Como Executar

1. Abra o terminal e navegue até o diretório do projeto.
2. Execute o script de mapeamento utilizando o comando:
   python Mapa_Estabelecimentos_05.py
3. Para visualizar o mapa, abra o arquivo `index.html` diretamente em seu navegador web.
